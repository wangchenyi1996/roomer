# roomer--该项目主要模拟蛋壳和自如APP页面，使用了Vue + Vuex + Vue-router + VantUI等技术

## 第一步
```
npm install
```

### 第二步 安装完依赖后，可直接运行
```
npm run serve
```

### 第三步打包
```
npm run build
```
